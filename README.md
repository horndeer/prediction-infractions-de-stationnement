##  Description des données

`issu de la description du concours`

Les données recouvrent les informations suivantes par contrôle :

Latitude et longitude de la zone géographique.
Horodatage (date et heure), échantillonnées à l’heure.
Taux d’infraction constaté dans la zone concernée entre l’horodatage actuel et le suivant.

Nous mettons à disposition un échantillon de ces données.
Nous proposons également d’autre données externes en entrée, accessibles en open data :

Conditions météorologiques (open data Météo France)
Le but est de prédire les futures infractions, en tenant compte de l’historique des infractions mais également en prenant en compte les facteurs externes (météo).

Il est proposé de considérer ce problème en tant que régression.
